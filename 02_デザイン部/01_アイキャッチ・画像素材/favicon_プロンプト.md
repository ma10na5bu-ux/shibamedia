## 共通指示

- **生成サイズ**: 768×768px（正方形）← ウォーターマーク除去用に大きめ生成
- **最終サイズ**: 512×512px（PILで右下クロップ＋リサイズ）
- **用途**: Webサイトのファビコン（ブラウザタブ・ブックマーク用アイコン）
- **注意**: 16×16pxに縮小しても認識できるシンプルさが必須

### ウォーターマーク対策
Gemini生成画像は右下にウォーターマークが付く。以下で対処：
1. 768×768pxで大きめに生成（余白を稼ぐ）
2. プロンプトでシルエットを「中央〜左上寄り」に配置指示
3. PILで右下をクロップ → 512×512にリサイズ

## 生成後の手順

1. Geminiで3パターンそれぞれ生成
2. 生成画像を保存 → PILで右下クロップ＋512×512リサイズ
   ```python
   from PIL import Image
   img = Image.open("favicon_raw.png")
   # 右100px・下100pxをクロップ（768→668程度）
   cropped = img.crop((0, 0, img.width - 100, img.height - 100))
   # 512×512にリサイズ
   resized = cropped.resize((512, 512), Image.LANCZOS)
   resized.save("favicon_512.png")
   ```
3. 目視確認（ウォーターマークが消えていること）
4. 社長に3つ並べて見せて選んでもらう
5. WordPress → 外観 → カスタマイズ → サイト基本情報 → サイトアイコン で設定
