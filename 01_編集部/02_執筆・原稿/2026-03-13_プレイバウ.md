散歩から帰ってきた直後、玄関でいきなり前足をぐーっと伸ばしてお尻を突き上げる。しっぽはブンブン、目はキラキラ。「え、今から？ 散歩終わったばかりなのに？」——柴犬飼いなら一度は見たことがあるこのポーズ、「プレイバウ」といいます。犬の「遊ぼう！」の合図として知られていますが、柴犬のプレイバウにはちょっと独特な事情がありそうです。この記事では、プレイバウの意味と、柴犬ならではの「あるある」シーンを紹介します。

## プレイバウってなに？

プレイバウとは、前足を低く伸ばし、お尻を高く上げるポーズのことです。

| 項目 | 内容 |
| :--- | :--- |
| ポーズ | 前足を床にペタッとつけて伸ばし、お尻と巻き尾を高く突き上げる |
| 意味 | 「遊ぼう！」という誘いのサイン。犬同士でも人に対しても使う |
| 表情 | 口が軽く開いて**柴スマイル**になっていることが多い |
| しっぽ | ブンブン振っている、または巻き尾がいつもより高い位置にある |

犬種を問わず見られるポーズですが、柴犬のプレイバウは**普段のツンデレ気質**とのギャップが大きいぶん、破壊力が段違いです。

## 柴犬プレイバウあるある5選

柴犬のプレイバウは「え、今？」「え、そこで？」というタイミングで発動することが多く、飼い主を困惑させつつも笑わせてくれます。

| あるある | 解説 |
| :--- | :--- |
| 散歩から帰った直後に玄関でプレイバウ | たっぷり歩いたはずなのに「まだ遊び足りない！」と全力アピール。散歩の興奮が余韻として残っているパターン |
| 飼い主がソファに座った瞬間を狙ってくる | こちらが「やっと座れた……」と思った0.5秒後に目の前でバウ。タイミングの読みが鋭すぎる |

「散歩帰りにプレイバウされると、疲れてるのに『もうちょっとだけ……』と付き合ってしまう」という飼い主は多いはずです。

## なぜ散歩の後にプレイバウするのか

散歩を終えたばかりなのに「遊ぼう！」と誘ってくる柴犬。矛盾しているように見えますが、いくつかの理由が考えられます。

| 理由 | 解説 |
| :--- | :--- |
| 興奮の余韻が残っている | 外で色々な匂いを嗅いで刺激を受けた興奮が、帰宅後にテンションとして発散される |
| 「楽しかった！」の表現 | 散歩が満足だったからこそ、その気分のまま飼い主と遊びたくなる |
| 帰宅＝安心できる場所 | 外では警戒モードだった柴犬が、家に戻ってリラックスした途端に甘えモードに切り替わる |
| 飼い主の注目を取り戻したい | 散歩中は外の世界に集中していたぶん、帰宅後に「飼い主と遊ぶ時間」を求めている |

散歩後のプレイバウは「帰ってきて安心した」「楽しかったからまだ遊びたい」という柴犬なりのポジティブな感情表現です。

## プレイバウされたらどうする？

プレイバウで誘われたとき、飼い主としてどう応えるかのヒントです。

| 対応 | ポイント |
| :--- | :--- |
| 少しだけ付き合ってあげる | 5分でOK。短時間でも「遊んでもらえた」という満足感が得られる |
| 同じポーズを返してみる | 飼い主がプレイバウを真似すると、柴犬が「わかってるじゃん！」と大喜びすることがある |
| おもちゃを一つ出す | 遊びの方向性を決めてあげると、お互い楽しめる |
| 毎回全力で応えなくてもOK | スルーしても柴犬は怒らない。「今日は無理だよ」と穏やかに声をかけるだけで十分 |

プレイバウは柴犬が飼い主に見せる数少ない「全力の甘え」です。毎回でなくても、たまに付き合ってあげると柴犬との信頼関係がさらに深まります。
